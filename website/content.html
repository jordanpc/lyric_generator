Project Overview

What's the Goal?

Everybody loves music and who wouldnâ€™t want to create their own song using technology (work smarter, not harder).The goal of our project was to generate song lyrics based on pop songs. To do this, we built a Long Short-Term Memory (LSTM) Recurrent Neural Network in Python with the Keras library package.

What data did we use?

last.fm & Genius.com API

It worked. However, we could not pull more than 5,000 tracks data due to (1) there might be a limit in the numbers of requests, and (2) we had to use last.fm & Genius.com which sometimes had different track names.

Kaggle Dataset - 380,000+ lyrics from MetroLyrics

We ended up using a static dataset in Kaggle. It contained over 380,000 songs data which is more than we can process.

What Machine Learning did we use?

Generative adversarial networks

Defintion: Generative adversarial networks are a class of artificial intelligence algorithms used in unsupervised machine learning, implemented by a system of two neural networks contesting with each other in a zero-sum game framework. They were introduced by Ian Goodfellow et al. in 2014.

What did we learn?

We learned that machine learning takes a lot of time as well as computer power, and we need to plan the architecture before we scale it. Otherwise, we might get unsatisfied results after all the effort. We also learned to utilize a virtual machine to help us train the model.
